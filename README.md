# Data for "Distributed Finite-Time Optimal Consensus Control for Strict-Feedback Nonlinear Multi-Agent Systems via Reinforcement Learning"

This repository contains the simulation data and plotting script used to generate all figures in the above paper.

## Files

- `data.mat` – MATLAB data file containing all simulation results (time series, states, errors, costs, weights, event-triggered data, etc.).
- `data_plot.m` – MATLAB script to load `data.mat` and reproduce Figures 1–8 in EPS format.

## Requirements

- MATLAB R2020a or later (earlier versions may also work).

## How to Run

1. Place `data.mat` and `data_plot.m` in the same folder.
2. Open and run `data_plot.m` in MATLAB.
3. The script will generate eight EPS files: `Fig1.eps` to `Fig8.eps`.
